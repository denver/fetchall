# fetchall

CLI tool to fetch all URLs from CSV file

Usage:
Build: `go build` and run exe. Or `go run main.go`

By default: default.csv will run. Alternatively pass in `-cvs=example.csv` flag

`go run main.go -csv=example.csv`
